Getting startedÂ¶

Install pptk.

>>> pip install pptk

Note

pptk requires 64-bit Python and is only pip install-able on versions of Python that have a corresponding pptk wheel file on PyPI.

In Python, generate 100 random 3-d points, and

>>> import numpy as np
>>> x = np.random.rand(100, 3)

Visualize.

>>> import pptk
>>> v = pptk.viewer(x)

Set point size to 0.01.

>>> v.set(point_size=0.01)
